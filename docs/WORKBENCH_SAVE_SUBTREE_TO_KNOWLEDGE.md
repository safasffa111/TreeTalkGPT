# 工作台问题树保存到知识仓库

## 目标

在工作台模块的逻辑树标题右侧加入无边框加号按钮。点击后进入“保存到知识仓库”状态，逻辑树中每个问题项目前方都会出现无边框加号按钮。点击某个问题项目前的加号后，会把该问题以及它后续的所有追问、框选上下文、回答内容保存为知识仓库根目录下的一个新文件。

## 交互

- 工作台逻辑树标题右侧新增 `＋` 按钮。
- 点击标题 `＋` 后，列表中每个问题项目前方渐进出现 `＋` 图标按钮。
- 该按钮动画复用知识仓库删除按钮风格：旋转、模糊、渐进/渐出。
- 再次点击标题 `＋` 退出保存状态，项目前方加号旋转模糊渐出。
- 快速点击标题 `＋` 时，通过队列只保留最后目标状态，避免动画重叠。

## 保存逻辑

点击某个问题项目前的 `＋` 后：

1. 以当前问题为根节点。
2. 收集该节点所有后代问题。
3. 复制这些问题节点中的：问题、框选原文、回答、显示中的回答、状态、附件、标注等字段。
4. 将选中的根节点 `parentId` 改为 `null`，形成一个新的独立问题树。
5. 过滤消息记录，只保留该子树相关的问题消息。
6. 在知识仓库根目录创建一个新文件。
7. 新文件的 `session` 保存完整子树快照，后续点击该知识文件时会恢复成独立学习树。
