# TreeTalk Desktop

TreeTalk Desktop 是一个面向深度学习与知识整理的本地优先桌面应用。它将连续追问组织成可回退的“学习调用栈”，并把问题树、回答、附件和知识关系沉淀到本地知识仓库中，帮助用户在复杂学习过程中保持清晰的上下文与结构。

> 项目正在进行桌面应用化和跨平台安装包完善，目标平台为 Windows 11 与 macOS。

## 核心方向

- **学习调用栈**：围绕主问题逐层追问、压栈和出栈，保存完整学习路径。
- **问题树与知识图谱**：可视化主问题、子问题、回答和知识点之间的关系。
- **本地知识仓库**：在设备本地保存学习会话、问题树、附件和整理结果。
- **附件与公式处理**：解析常见文档，支持数学公式和化学公式渲染。
- **桌面端体验**：历史记录、错误日志、无边框窗口、动画队列和中央状态调度。
- **跨平台分发**：计划提供 Windows 11 安装包，以及适用于 Intel 和 Apple Silicon 的 macOS 安装包。

## 产品目标

TreeTalk Desktop 希望解决普通对话式学习中容易出现的几个问题：

1. 连续追问后忘记原始问题和当前学习位置。
2. 子问题之间缺少明确的父子关系和回退路径。
3. 对话结束后难以形成可以复习和继续维护的知识结构。
4. 学习内容、附件、问题树和知识图谱分散在不同工具中。

产品的核心流程是：

```text
提出主问题 → 针对不理解的内容继续追问 → 完成当前分支后返回上层
→ 保存完整问题树 → 生成知识点总结与知识图谱 → 沉淀到本地知识仓库
```

## 数据与隐私

项目采用本地优先设计。学习会话、知识仓库和附件原则上保存在用户设备上。仓库中不得提交 API Key、账号凭据、用户附件或个人知识库数据。
