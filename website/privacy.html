Privacy

隐私说明

本页面说明 TreeTalk 官网与桌面应用在当前公开版本中的基本数据处理方式。TreeTalk 采用本地优先设计，不提供集中式用户账户与云端作品库。

官网

官网是托管在 GitHub Pages 上的静态网站，不包含广告、用户画像脚本或第三方统计代码。为了显示最新版和提供直接下载，浏览器会向 GitHub 的公开 Release API 请求版本、文件大小和下载次数。

桌面应用的本地数据

标签页、问题树、知识或记忆内容、附件索引以及应用配置主要保存在用户自己的设备中。卸载、移动或清理本地文件前，请自行备份重要内容。

AI 服务

当你使用 API 模式或浏览器桥接模式时，当前问题、附件以及你主动引用的上下文会发送给你选择的模型服务商或 ChatGPT 网页服务。相关数据如何被处理，取决于对应服务商的隐私政策与账户设置。

API Key

API Key 用于从本地应用连接模型服务。请不要在反馈截图、公开仓库或共享配置中暴露密钥。

问题反馈

通过 GitHub Issues 提交反馈时，内容会按照 GitHub 仓库的可见性公开展示。请勿上传真实密钥、私人作品全文或包含敏感信息的日志。
